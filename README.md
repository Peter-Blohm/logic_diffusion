# Code for the ICLR 2026 paper "Composition of Pretrained Diffusion Models: A Logic-Based Calculus"

This repository contains implementations and experiments adapted from several prior works.  

## Experiments

All experiments are located in the [`experiments/`](./experiments) directory.  
For instructions on how to run them, please take a look at the **README** files within each experiment folder or the provided run scripts.

## Sources

Our experiments are adapted from the following repositories:

- [Compositional Sculpting](https://github.com/timgaripov/compositional-sculpting)
- [Super Diffusion](https://github.com/necludov/super-diffusion)
- [FKC Diffusion](https://github.com/martaskrt/fkc-diffusion)
